# BCS Web

This is the React + TypeScript + Vite frontend for the **BCS** project.

The project uses Vite for fast development, React with TypeScript, and ESLint for linting.

---

## Tech Stack

- Vite
- React
- TypeScript
- WebRTC

---

## Getting Started

### Install dependencies

```bash
npm install

or

yarn

or

pnpm install

Run development server

npm run dev

The application will be available at the URL printed in the terminal
(default: http://localhost:5173

).
Build for production

npm run build

The production build will be generated in the dist directory.
Preview production build

npm run preview

Linting

npm run lint

npm run dev – start development server

npm run build – build for production

pm run preview – preview production build

npm run lint – run ESLint

```
